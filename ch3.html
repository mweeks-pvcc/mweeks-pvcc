Terms

  1. Operating System
    • is the software that controls the overall operation of a computer
  2. Job Queue
    • is a storage organization in which objects(in this case,jobs) are ordered in FIFO
  3. FIFO
    • First In, First Out
  4. Interactive Processing
    • allowing a program to be executed to carry dialog with the user through remote terminals
  5. Time-Sharing
    • operating systems that provided service to multiple users at the same time
  6. Multipleprogramming
    • Time-Sharing in which time is divided into intervals and then the execution of each job is restricted to only one interval at a time
  7. Application Software
    • consists of the programs for performing tasks particular to the machine's utilization
  8. System Software
    • is divided into two categories, one is the operating system itself, and the other consists of software units collectively known as Utility Software
  9. Utility Software
    • programs for preforming activities that are fundamental to computer installations but not included in the operating system
  10. Graphical user interface
    • system in which objects are manipulated such as files, and programs are represented pictorally on the display as icons
  11. (GUI)
    • short for Graphics User Interface
  12. Shell
    • older user interface, that commmunicated with users through textual messages using a keyboard and monitor screen
  13. Kernel
    • the internal part of an operating system, consists of software programs that preform basic functions required by the computer installation
  14. Device Driver
    • are the software units that communicate with the controllers (or at times,directly with peripheral devices) to carry out operations on the peripheral devices attatched to the machine
  15. File Manager
    • software componet whose job is to coordinate the use of the machine's mass storage facilities
    Virtual Memory
    • the product of the memory manager shuffeling pages back and fourth between main memory and mass storage, so that the pages that are needed are actualy present in the 4GB of main memory, this "fictional" memory space
  16. Boot Strapping
    • is preformed each time a computer is turned on, procedure that transfers the operating system from mass storage(where it is permanetly stored) into main memory(which is essentially empty when the machine is first turned on)
  17. Rom
    • read-only memory, memory that can be read but can not be altered, special nonvolatile cells constructed from a small portion of the main memory
  18. Deadlock
    • the condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to the other
  19. Spooling
    • technique of holding data for output at a later but more convienient time
  20. Super User or Administrator
    • has highly privileged access to the operating system by identifying him-or herself as the Administrator ( usually by name and password) during the login procedure, can alter settings within the operating system, modify critical software packages, adjust the privileges granted to other users, and preform a variety of other maintenance activities that are denied normal users
  21. Multi-factor authentication
    • multiple factors used to grant authentication, such as passwords, biometrics, special access code, none are entirely fool proof
